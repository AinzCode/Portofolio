# Portfolio Website

Modern portfolio website with red and black theme.

## Features

- Red and black color scheme
- Full-screen hero section with background image
- Scroll-snap navigation (sections snap when scrolling)
- Dark mode toggle with light/dark themes
- Responsive design for all devices
- Smooth animations and transitions
- TypeScript for type safety
- Font Awesome icons
- Contact form
- Mobile navigation menu

## Functions

- Auto-scroll snaps to each section
- Dark mode saves preference to localStorage
- Typing animation on hero name
- Parallax effect on hero background
- Hover effects on buttons and cards
- Form validation on contact form
- Smooth scroll to sections on navigation click
- Active navigation link highlighting
- Mobile menu toggle

## Setup

Install dependencies:
```bash
npm install
```

Start development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Technologies

- Vite
- TypeScript
- Tailwind CSS
- Font Awesome
